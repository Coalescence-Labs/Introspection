# Design Resources

This folder contains visual inspiration, reference materials, and design assets for the Introspection project.

## Purpose

Collect and organize design inspiration that aligns with our Awwwards-level visual approach:
- Bold, oversized typography
- Generous whitespace and breathing room
- Minimal, impactful UI components
- Popping accent colors
- Premium, polished aesthetic

## Organization

### Naming Convention

Use descriptive names that indicate the source or aspect being referenced:

- `introspection-design-inspo-[number].jpg` - General inspiration
- `typography-example-[source].png` - Typography references
- `layout-[description].jpg` - Layout and spacing examples
- `color-palette-[name].png` - Color scheme references
- `animation-[interaction].gif` - Motion and interaction examples
- `component-[name].png` - Specific UI component references

### Categories

**Typography**
- Bold, oversized text treatments
- Font pairings
- Type scales in action

**Layout & Spacing**
- Generous whitespace examples
- Centered, focused compositions
- Vertical rhythm demonstrations

**Color**
- Minimal palettes with impact
- Accent color usage
- High contrast examples

**Interactions**
- Button states and hover effects
- Transitions and animations
- Micro-interactions

**Overall Inspiration**
- Full page screenshots from Awwwards sites
- Linear, Stripe, Arc Browser, etc.
- Any design that embodies "less is more, what remains is BOLD"

## Usage

When designing or implementing features:
1. Review relevant files in this folder
2. Extract key principles (spacing, scale, color usage)
3. Adapt inspiration to Introspection's specific needs
4. Add new references as you discover inspiring examples
